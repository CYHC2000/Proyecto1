📋 Descripción del Proyecto
Este es un portafolio web personal que muestra proyectos y habilidades en diversas áreas tecnológicas. El sitio está desarrollado utilizando HTML puro y presenta una estructura organizada en múltiples páginas interconectadas.

🎯 Propósito
El objetivo de este portafolio es:

Exhibir proyectos realizados en diferentes especialidades tecnológicas

Proporcionar información profesional sobre el desarrollador

Ofrecer un medio de contacto para posibles colaboraciones o empleo

🛠️ Tecnologías Utilizadas
HTML5: Estructura semántica del sitio web

Navegación entre páginas: Sistema de rutas relativas

Imágenes: Uso de recursos visuales para presentar proyectos
🗂️ Secciones del Portafolio
Página Principal (index.html)
Presentación personal

Metas profesionales

Vista previa de proyectos por categoría

Formulario de contacto

Páginas Especializadas
Inteligencia Artificial: Chatbots, identificadores de imágenes, agentes especializados

Análisis de Datos: Proyectos de análisis estadístico y estudios de datos

Dashboards: Visualizaciones de datos e interfaces de monitoreo

Proyectos Web: Soluciones web como sistemas de citas y pedidos

✨ Características
Diseño responsivo: Adaptable a diferentes dispositivos

Navegación intuitiva: Menú de navegación consistente en todas las páginas

Presentación de proyectos: Cada proyecto incluye título, imagen y descripción

Sección de contacto: Formulario y enlaces a redes profesionales

Contenido adicional: Noticias relacionadas en algunas secciones

🚀 Cómo Ejecutar el Proyecto
Clona o descarga el proyecto en tu equipo local

Abre el archivo index.html en cualquier navegador web moderno

Navega entre las diferentes secciones usando el menú de navegación

📞 Información de Contacto

Formulario de contacto para solicitudes

Enlaces a perfiles profesionales (GitHub, LinkedIn, OCC, WhatsApp)
